# 預期圖片清單

此目錄應包含以下檔案，被 [../engine-room-inspection.csv](../engine-room-inspection.csv) 第 6 欄引用：

| 檔名 | 出現於 step | 內容建議 |
|---|---|---|
| `engine-hood.jpg` | 1 | 引擎蓋拉桿位置或開啟後示意 |
| `oil-dipstick.jpg` | 2 | 機油尺正確油位範圍標示 |
| `coolant.jpg` | 3 | 副水箱 MIN/MAX 標線特寫 |
| `dashboard.jpg` | 6 | 儀表板警示燈圖例 |

開發階段可以用任何小尺寸 jpg 檔（甚至 1×1 純色）暫代，重點是檔名要對得上。
